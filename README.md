# jailrun-hub

A curated collection of versioned Ansible playbooks for [jrun](https://github.com/hyphatech/jailrun): runtimes, databases, and services.

| Playbook | Description |
|---|---|
| [`imagor`](https://github.com/hyphatech/jailrun-hub/tree/main/playbooks/imagor/latest) | Imagor image processing service (deps + install). |
| [`nginx`](https://github.com/hyphatech/jailrun-hub/tree/main/playbooks/nginx/latest) | Nginx web server with a basic default config. |
| [`postgres16`](https://github.com/hyphatech/jailrun-hub/tree/main/playbooks/postgres/16) | PostgreSQL 16 database server. |
| [`redis`](https://github.com/hyphatech/jailrun-hub/tree/main/playbooks/redis/latest) | Redis in-memory data store. |
